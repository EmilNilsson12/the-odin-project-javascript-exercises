The function removeFromArray(inputArr, ...valuesToFilter)


Given two or more parameters where the first parameter is an array, and the rest are values which shall be filtered out. Returns an array of the remaining values.

removeFromArray(["hey", 2, 3, "ho"], "hey", 3));

The function call above would return the following array

[2, "ho"]

The function takes type into account

removeFromArray([1, 2, 3], "1", 3);

The function call above would return the following array

[1, 2]
